# 设备选择功能说明

## 新增功能

程序现在启动时会显示一个设备选择对话框，让您可以选择：

### 1. 相机选择 (Camera Selection)
- **OpenCV Camera**: 普通的 USB 摄像头或网络摄像头
  - 自动扫描系统中所有可用的摄像头（索引 0-9）
  - 显示为 "OpenCV Camera (index X)"
  
- **Basler Camera**: 工业 Basler 摄像头
  - 自动检测连接的 Basler 相机
  - 显示为 "Basler Camera"

### 2. 串口端口选择 (Serial Port Selection)
- 自动列出所有可用的串口端口
- 显示格式：`COMx - 设备描述`
- 优先自动选择包含 USB 或 Serial 的端口

## 使用步骤

1. **启动程序**
   - 运行 `python main.py` 或双击 `OpenMicroManipulator.exe`

2. **选择设备**
   - 在相机列表中选择要使用的相机
   - 在下拉菜单中选择微操作台连接的串口

3. **刷新设备**
   - 如果设备未显示，点击 "Refresh" 按钮重新扫描
   - 确保设备已正确连接到计算机

4. **开始运行**
   - 点击 "Start" 按钮启动程序主界面

## 故障排除

### 相机未显示
- 检查相机是否正确连接
- 尝试点击 "Refresh" 按钮
- 对于 OpenCV 相机，确保没有其他程序占用该相机

### 串口端口未显示
- 检查微操作台控制器是否通过 USB 连接到电脑
- 确认已安装正确的驱动程序
- 在设备管理器中查看端口号

### 连接失败
- 确认选择的端口和设备正确
- 检查物理连接
- 重启程序重新选择

## 技术细节

### 自动扫描的端口类型
- Windows: COM 端口（USB 转串口、蓝牙串口等）
- Linux: /dev/ttyUSB*, /dev/ttyACM* 等

### 相机检测机制
- OpenCV 相机：尝试打开每个摄像头索引并捕获测试帧
- Basler 相机：使用 Pylon SDK 枚举 USB 设备

## 高级配置

如需固定使用特定设备而不想每次选择，可以编辑 `main.py`：

```python
# 在 main() 函数中直接设置
camera_type = 'opencv'  # 或 'basler'
camera_idx = 0          # 相机索引
port = 'COM6'           # 串口号
```

但这会跳过设备选择对话框。
