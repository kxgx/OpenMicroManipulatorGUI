# OpenMicroManipulator 打包说明

## 打包成功！

可执行文件已生成在 `dist` 文件夹中：
- **文件名**: `OpenMicroManipulator.exe`
- **大小**: 约 231 MB

## 使用方法

### 方式一：直接运行
双击 `dist\OpenMicroManipulator.exe` 即可运行程序。

### 方式二：使用批处理脚本
以后如果需要重新打包，只需双击运行 `build.bat` 文件即可。

## 注意事项

1. **首次运行**: 生成的 .exe 文件较大（约 231MB），因为包含了所有依赖库（PySide6、OpenCV、NumPy 等）

2. **分发**: 可以将 `OpenMicroManipulator.exe` 单独分发给其他用户，无需安装 Python 环境

3. **隐藏导入警告**: 构建过程中有一个警告 `Hidden import "scipy.special._cdflib" not found!`，这通常不会影响程序运行，因为该模块很少使用

4. **配置文件**: 
   - `build.spec` - PyInstaller 配置文件
   - `build.bat` - 一键打包脚本

## 如需自定义打包选项

编辑 `build.spec` 文件可以修改：
- 应用程序名称 (`name` 参数)
- 是否显示控制台窗口 (`console` 参数)
- 包含的额外文件和依赖
- 图标文件 (`icon` 参数)
